# Typescript 实现爬虫项目

- 安装

```bash
git clone https://github.com/TJS-95/crowller.git
```

- 下载依赖

```bash
npm install
```

superagent 爬取页面数据(类似 axios 发请求)

```bash
npm i superagent --save
npm i @types/superagent --save-dev
```

cheerio 解析返回的 html，类似 jQuery

```bash
npm i cheerio --save-dev
```

项目采用单例模式和组合模式
